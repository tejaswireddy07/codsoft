# Task 3: Iris Flower Classification

## Objective
To classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

based on sepal and petal measurements.

## Dataset
- IRIS.csv
- Target column: species

## Algorithm Used
- Logistic Regression

## Tools & Libraries
- Python
- Pandas
- Scikit-learn

## Steps Performed
1. Data loading
2. Label encoding
3. Train-test split
4. Model training
5. Model evaluation using accuracy, confusion matrix, and classification report

## Result
The model classifies Iris flower species with high accuracy.
